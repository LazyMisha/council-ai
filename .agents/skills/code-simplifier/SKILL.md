# Code Simplifier

## Instructions

- Remove duplication.
- Remove unused imports, props, and helpers.
- Simplify overcomplicated components.
- Prefer small safe refactors.
- Preserve UI and behavior.
- Keep the MVP minimal and chat-like.
- Do not add dependencies.
- Run `npm run quality` after changes if available.
